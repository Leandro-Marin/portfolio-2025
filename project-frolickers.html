FROLICKERS

2024

Frolickers is a sustainable fashion brand from England. The purpose of this project was to create a website for the brand that reflects Frolickers' values, which are focused on producing high-quality garments that last, demonstrating the brand's commitment and responsibility to the environment and consumers. The project was carried out at Figmenta.
